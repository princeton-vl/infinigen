Material Comparison UI

Run

Compare specific outputs: python scripts/integration_v2/compare.py outputs/v1 outputs/v2

Pick from a directory (landing page): python scripts/integration_v2/compare.py --scan-dir outputs

Views & Shortcuts

  • Side-by-side: Columns per version.
  • Toggle: One stack; press t to cycle versions.
  • Press s to jump back to side-by-side.

Controls

  • Size slider (100–600px) and mode toggle in the top bar.
  • ⚙ Filters: category (distribution/displacement), pass (rgb/surface-normal), renderer (cycles/eevee), align placeholders on/off.
  • Materials: select/deselect, include per-version, union vs intersection.
  • Selections persist when switching views.

Layout

Images are grouped into Random seeds (seed 0–3) and Displacement tests (BUMP/DISPLACEMENT/DISPLACEMENT_AND_BUMP). Missing renders show a dark “Missing” placeholder when alignment is on.
